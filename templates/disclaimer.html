Overview

This Bitcoin 2 Web Wallet is an open-source interface designed to interact with the Bitcoin 2 blockchain. Unlike traditional web wallets, your private keys are fully controlled by you, enabling direct importation of funds into any Bitcoin 2 wallet that supports standard key formats.

For the original Bitcoin 2 wallet and how to get started with btc2, please visit https://bitcoin2.org.

Important Security Warning

⚠️ Start with Small Amounts: We strongly recommend testing the system with small amounts first before conducting larger transactions. This allows you to familiarize yourself with the wallet's features and verify its functionality with minimal risk.

Technical Considerations

  • Balance Loading: Please note that balance updates may take some time to appear. The system needs to sync with the blockchain and verify transactions, so patience is sometimes required when checking balances or transaction status.
  • Network Synchronization: The wallet needs to maintain synchronization with the Bitcoin 2 network. This process can sometimes cause slight delays in displaying up-to-date information.

Staking Information

This web wallet includes auto-staking functionality with the following characteristics:

  • Staking continues even when you're logged out or your browser is closed
  • Larger balances typically receive more frequent staking rewards
  • During staking reward processing, coins may be locked for approximately 100 blocks (~1 hour and 40 minutes)
  • You receive 2/3 of staking rewards plus transaction fees for blocks you stake
  • 1/3 of staking rewards are automatically allocated to support ongoing development and maintenance
